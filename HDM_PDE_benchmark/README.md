# HDM_PDE_benchmark

Source Drive folder:

- `HDM_PDE_benchmark`: https://drive.google.com/drive/folders/1AxKuYHunx2506MMROe6MGMxYFm9vk4qa

## Notes

The Drive folder contains generated figures, logs, and checkpoint-related outputs. Those are referenced in `MANIFEST.md` rather than copied byte-for-byte, because the available Drive connector exposes many of them as extracted text or truncated large text rather than reusable raw bytes for GitHub upload.
